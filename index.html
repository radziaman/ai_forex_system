AI‑Powered Forex Scalper cBot

Overview

This cBot runs on cTrader Cloud and uses a machine‑learning model to generate sniper‑entry signals, risk‑managed position sizing and automated profit‑taking.

Features

  • Real‑time market data via cTrader Cloud API
  • ML.NET random‑forest model with technical‑indicator features
  • Dynamic risk manager (margin, volatility, liquidity checks)
  • Backtesting engine with Sharpe ratio & max drawdown
  • GitHub‑Pages dashboard for monitoring
