🤖 AI Forex Scalper cBot

Machine Learning Powered Trading on cTrader Cloud

📊 Overview

This AI-driven cBot runs on cTrader Cloud and uses machine learning to generate sniper-entry signals with institutional-grade risk management and automated profit-taking.

🧠 AI Strategy

  • FastTree Regression Model
  • Technical Indicator Features (RSI, MACD, Bollinger)
  • Dynamic Confidence Scoring
  • Pattern Recognition

🛡️ Risk Management

  • Liquidity-based Position Sizing
  • Margin Validation
  • Price Limit Checks
  • Dynamic Stop-Loss

📈 Backtesting

  • Historical Bar Simulation
  • Sharpe Ratio Calculation
  • Max Drawdown Analysis
  • Trade Record Export

☁️ cTrader Cloud

  • Real-time Market Data
  • Order Execution API
  • Account Integration
  • Sniper Entries
